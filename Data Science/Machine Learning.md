# Machine Learning

Machine Learning ist ein Teil der künstlichen Intelligenz und steht für die künstliche Generierung von Wissen und Erfahrung. Dabei handelt es sich um eine Ansammlung von mathematischen Methoden zur Mustererkennung. Diese mathematischen Methoden sind u.a. Algorithmen, die es möglich machen, viele alltägliche und auch spezielle Probleme zu lösen. Einfach gesagt, sollen Maschinen so trainiert werden, dass sie selbstständig lernen und entsprechend daraus handeln können ohne das ein menschliches zutun vonnöten ist. 
Eine Technologie, welche aktuell immer mehr an Bedeutung im Zusammenhang mit Machine Learning erhält, ist die sogenannte Blockchain. Im Kapitel xy wird diese Technologie nochmal genauer aufgegriffen. 
Damit Machine Learning erfolgreich genutzt werden kann, bedarf es einiger zu erfüllender Rahmenbedingungen, welche da die Funktionen für die Datenaufbereitung wären, einfach und komplexe Algorithmen, Prozesse für die Automatisierung und Wiederholung, Skalierbarkeit und Ensemble-Modellierung.
